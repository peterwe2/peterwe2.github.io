讀書 - Soft Skills 軟實力

Posted on 2020-05-17 Edited on 2020-08-16

Soft Skills 軟實力｜軟體開發人員的生存手冊 (Soft Skills: The software developer’s life manual)

ISBN: 9864766384

內容摘要

在這本書裡面，主要介紹了軟體開發人員的職涯中，可以以什麼樣的方法與技巧去面對可能會遭遇到的非技能相關的主題。同樣的，我們必須越早知道除了coding之外，還需要培養一些技巧去優化”人生”這個程式，在讀完之後，紀錄一下一些整理的重點和想法。

關於職涯

以前上學的期間環境跟上班的環境截然不同，在工作的時候遇到想都沒想過的問題是很正常的，不過要如何能夠迎刃而解這些問題，才是真正重要的，比如自己要一輩子當碼農嗎? 有辦法突破嗎? 又或者怎樣可以提升自己的生產力，為工作帶來更高的價值? 很認同作者表示履歷不該只是一張紙，當有機會為 open source 貢獻的時候、自己的 side project 做的很有意思，這些作品集才能真正的體現出經歷，而不以時間說明自己的工作經驗。

四大主題

除了專業技能之外，還必須要認真的經營未來會跟著自己一輩子的: 身體健康、社交生活、財務管理、學習能力

  1. 關於健康，曾經在網路上看過一個公式: 成就 = 身體健康 * 你付出的一切，所以保持運動的習慣、正常的飲食，讓身體能夠有好精神，在做任何事情都事半功倍，也不忘要維持mental health，我認同這個為第一重要。每天都可以慢跑個半小時，順便舒緩一天上班的疲憊，增加多巴胺，讓腦袋更快樂!

  2. 關於社交生活，跟家人聊天，找同學朋友打屁，維繫人與人之間的感情也是特別重要的，在講話的過程中，也可以訓練說話邏輯的能力，同時訓練溝通的能力，這也會為自己帶來優勢，避免成為工程師中的孤狼。

  3. 關於理財，有句話說 你不理財，財不理你，或許成為軟體工程師不能讓你發大財，但是幫自己做好財務的管理，是一個必須養成的習慣，比如記帳可以記錄過去花了多少錢，在做檢討的時候，可以避免自己再次的衝動消費，比如投資，當累積了一筆閒錢，能夠拿去讓”錢自己賺錢”，有計畫的花錢與存錢，才是真正的讓薪水變成喜歡的樣子。

  4. 關於學習能力，活到老學到老，尤其對於軟體工程師特別重要，在開發的過程中很容易會遇到目前在用的工具已經過時或是需要更新等等，那勢必要因應趨勢去了解並學習新的工具，即使不是要追趕最新的項目，擁有一套高效的學習套路，在任何領域和時機都能夠派上用場，比如碰到一個從前都沒有遇到過的領域，那麼要如何讓自己短時間上手? 上手之後又要如何熟練?

結語

書中還有很多topic沒有在這邊提出來，很推薦大家把這本書看完，即使你會覺得這些東西好像是廢話，不過對於工作才1年的菜雞－我來說，在目前的職涯遇到一些迷惘的時候，這本生存手冊確實給了不少實用的建議。

2019 Review, 從學生到職場

Posted on 2019-12-31 Edited on 2020-08-16

2019 Review, 從學生到職場

1. 前言

時間過得真快, 轉眼已經2019年底
這一年真正的從學校步入社會
離開無憂無慮的學生舒適圈
開始現實社會中生存的菜雞生涯
工作了8個多月, 在這裡紀錄今年發生的一些回顧和心得

2. 工作

找工作

在退伍後剛好過年, 在2019的2月底才開始找工作
因為研究所的時候主要在做的領域就是 LTE/ 5G 相關, 在找職缺的時候也就投遞許多相關的職位
同時也將目標職缺放廣到C/ C++, Linux, network且軟體或韌體等等都可以嘗試
期間也是不停的蒐集各種相關的面試經驗, 筆試題目, 面試題目, 還要不停的修改與完善個人簡歷
面完一間還要檢討每次哪裏可以講的更好更順,
在面試期間的中後期開始還要考慮哪些機會必須先放棄以及哪些必須要想個理由請公司方等待
尤其是在收到公司來的邀約信還需要安排時間跟舟車勞頓
後來面試了21個, 有7個offer get, 命中率偏低 QQ
在面試了大概6個禮拜後, 結束求職之旅, 終於找到了我的第一份工作!

社畜人生

關於開始上班之後
就從第一天到職的時候說起吧
最大的不一樣在於
以前還是學生的時候
每到一個新環境就會有前輩為新人迎新的活動等等
而在公司這, 就少了那份歡迎的感覺
畢竟大家來到公司也只是為了討生活

不過或許是我們這個部門比較特別, 大家很”避暑”沒什麼閒聊等等
(大家人都很nice, 也很強, 只是不愛聊天XD)
但也不完全是缺點, 不會有那種大家都在幹嘛所以我也必須幹嘛的壓力
只是社交生活大概就等於0 haha
其餘的時候都跟研究所的時候滿像的
而由於我們這個部門的產品在公司中算是新的
目前也沒有什麼 deadline 的壓力, 完全就是 research 階段
也就不太需要加班,真995(早上9點到公司, 每天工時9小時, 每周工作5天).
在上班時間主管不太會去要求必須做什麼, 主要任務就是把現有的文檔跟 source code摸熟
可能是要做的東西太新, 主管大部分的時候真的不太派其他事情給我, 大多都是幫忙測試東西
這半年多來, 認為自己在找新東西來研究的能力有所提升, 也拜讀了很多軟體工程前輩的優質文章
找了不少有趣的 topic, 之後整理整理, 希望2020可以找時間來好好研究

3. 生活

在高雄南漂了6年(大學4年+研究所2年), 完成了學業
2018/10 一拿到畢業證書就先入伍受訓4個月
(許多長輩好像挺好奇4個月能幹嘛, 改天再發一篇仔細的回顧, 之前根本想都不想回想XD)
然後才真正的搬回到北部老家
其實不太習慣
比起大學的時候, 在研究所的時候更多時候都是自己一個人相處
從實驗室回到租屋處就是想做什麼就做什麼, 不太會有人管,
時間上的規劃更為自由
回到家裡就是要顧慮家人的感受, 比如電動不能打太晚, 要一起分擔很多家事, 要跟大家一起吃飯等等
(而且特別想念在高雄的許多宵夜….)
但整體來說過著挺規律的生活

4. 目標

“投資自己還是有最大的報酬率”
2019的時候沒立什麼flag
唯一一個大目標就是找到一份work life balance 的工作
很幸運的自己認為是達到了
但是看了很多大前輩的老人言: software engineer 就是必須活到老學到老, 學海無涯
在上班之後有一些小目標但是沒有達到:

  • 看完兩本書並寫總結,
  • 完成資料結購的複習並寫下學習紀錄
  • 每周固定運動至少3天
    只有完成: 學習coursera 的 learning how to learn
    總歸來講還是需要培養自己的自制力 -> attention management
    還有一些壞習慣必須剔除: 晚睡, 划太多手機看雜訊, 泡在影集中

希望2020可以完成:

  1. 固定看書寫下總結的習慣: mindmap+ 知識沉澱
  2. 每周固定運動的習慣
  3. 寫一點 leetcode 活動腦袋
  4. 培養 writing skills: 能夠做出高質量的文檔
  5. finance management: 開始練習”投資”
  6. 學python
  7. 經營 github issue blog (這是第一篇haha), 寫雜記, 技術文章,
  8. 完成一兩個 side project

5. 總結

“機會是留給準備好的人”
現在的自己還沒辦法成為想要成為的那種人
看了很多一流大公司的前輩分享生活與工作
確實是很羨慕, 也知道了自己還有很多的不足
希望自己可以在未來一年中, 每一天都能更加進步
